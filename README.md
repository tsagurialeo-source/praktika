# C++ Проект: Поиск максимального элемента в массиве

## Описание
Программа на C++ для поиска максимального элемента в массиве с интерактивным пользовательским вводом.

## Функционал
1. **Ввод количества элементов** - пользователь указывает размер массива
2. **Поэлементный ввод** - последовательный ввод каждого элемента
3. **Поиск максимума** - использование функции `findMax` для вычисления
4. **Вывод результатов** - отображение массива и найденного максимума
